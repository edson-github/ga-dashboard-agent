# GA Dashboard Agent

## Sobre o Projeto

Este projeto consiste em um agente de engenharia de dados especializado em IA, projetado para analisar dados do Google Analytics.

## Funcionalidades

O agente é capaz de:

1.  Receber arquivos CSV contendo dados do Google Analytics.
2.  Criar dashboards detalhados explicando a performance dos usuários.
3.  Fornecer análises detalhadas do comportamento do usuário por:
    *   Origem/Mídia (Source/Medium)
    *   Outros eventos relevantes no site

## Objetivo

O objetivo principal é transformar dados brutos em insights acionáveis através de explicações detalhadas e visualizações sobre o comportamento dos usuários.
